# 🧪 Unit Test Document – Table time

## Inleiding
Binnen het project **Table time** zijn geautomatiseerde unit tests opgesteld om kritieke functionaliteiten rondom reserveringen te valideren.  
De tests zijn geschreven met **Vitest** en richten zich op front-end business logic in de Vue-applicatie.

Elke test is expliciet gekoppeld aan een **concrete user story uit Sprint 0**, zodat duidelijk is welke functionele eisen door middel van tests worden afgedekt.

---

## Testomgeving

- **Testframework:** Vitest  
- **Programmeertaal:** JavaScript  
- **Front-end:** Vue + Tailwind  
- **Testtype:** Unit tests  
- **Locatie tests:** `/utils/*.test.js`
